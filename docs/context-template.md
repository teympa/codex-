# Context Template

以下は `context.md` を更新するときの最小テンプレート。

```md
# Context

## Objective

このワークスペースの大目的を書く。

## Current Focus

- いま進めている主対象
- 直近で終わらせたいこと

## Active Deliverables

- 現在の主要ファイル

## Source Of Truth

- Notion:
- GitHub:
- context.md:

## Working Rules

- 作業中の基本ルール

## Immediate Next Steps

1. 次にやること
2. その次にやること

## Decision Log

- YYYY-MM-DD: 判断内容

## References

- 参照すべきローカル文書

## Handoff

- 次回起動時の入口
```

## Quick Update Template

作業の最後に最低限これだけ更新する。

```md
## Current Focus

- いまの主対象

## Immediate Next Steps

1. 次の1手

## Decision Log

- YYYY-MM-DD: 今回決めたこと

## Handoff

- 次回はここから始める
```
